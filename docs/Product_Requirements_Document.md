# MinTik 产品需求文档 (简易版)

## 1. 产品简介
**MinTik** 是一款专为 macOS 用户打造的智能专注与健康助手。它不仅仅是一个计时器，更是一个能够感知你工作节奏的智能伴侣。MinTik 旨在帮助用户在保持高效工作的同时，通过科学的休息提醒，避免过度疲劳，实现工作与健康的完美平衡。

## 2. 核心功能

### 2.1 智能专注计时
- **自动感知**：无需手动操作，MinTik 能智能识别你的工作状态。
- **沉浸体验**：极简的计时界面，让你专注于当下的任务，不受干扰。

### 2.2 动态活动热力图
- **可视化记录**：通过精美的分钟级热力图，直观展示你过去一小时的投入程度。
- **成就感反馈**：看着格子的颜色随着专注度加深，获得即时的正向反馈。

### 2.3 智能休息提醒
- **健康守护**：当监测到长时间连续工作时，MinTik 会温柔地提醒你休息。
- **防倦怠机制**：基于实际活动量的算法，比固定闹钟更懂你的疲劳度。

### 2.4 极简菜单栏驻留
- **零打扰**：安静地驻留在 macOS 菜单栏，随时待命，不占用宝贵的屏幕空间。
- **一键呼出**：需要时一键唤起，查看状态或调整设置。

## 3. 产品优势

### 3.1 极致的无感体验
MinTik 的设计哲学是"不打扰"。它不需要你刻意去点击"开始"或"结束"，而是默默地在后台为你记录每一分努力。

### 3.2 隐私安全第一
- **本地运行**：所有数据完全存储在你的 Mac 本地。
- **无云端上传**：我们不收集、不上传任何用户行为数据，彻底消除隐私顾虑。

### 3.3 科技美学设计
- **原生质感**：完美融入 macOS 生态的 UI 设计。
- **炫酷视觉**：深色模式、霓虹光效与玻璃拟态的结合，让工具也成为一种视觉享受。

## 4. 适用人群
- **创意工作者**：设计师、作家、开发者等需要长时间沉浸工作的用户。
- **学生群体**：需要管理学习时间，保持高效专注的学生。
- **远程办公者**：在家办公需要自我监督和节奏管理的职场人士。

## 5. 产品愿景
让每一次专注都更有价值，让每一次休息都恰到好处。MinTik 致力于成为 macOS 平台上最懂你的专注伴侣。
